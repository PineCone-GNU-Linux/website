PineCone GNU/Linux

A Linux distribution based on Void Linux. With it's focus on Security and Hardening.

Download/Install

Key Features:

  • 🔒 AppArmor + hardened_malloc — Protection from exploits and leaks.
  • PineFarms — PineCone's own XBPS repositories.
  • Pledging — Restrictions of process' priveleges.
  • Xfce4 — A light, yet still modern Desktop Environment on the classic X11.
  • Hardened Linux — The Linux kernel, with security patches.
  • LibreWolf - A fork of Firefox, focusing on maximum privacy

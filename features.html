What makes PineCone unique?

Security out of the box

  • AppArmor — Control of the processes.
  • hardened_malloc — Protection from heap-corruptions, from GrapheneOS project.
  • linux‑hardened — Kernel, modified from GrapheneOS project.
  • libpledge.so — Port of the pledge function from OpenBSD, working automatically and through seccomp.

Minimalism and control

  • Void Linux — rolling release, runit, and a very fast package manager XBPS.
  • Xfce4 — Classic X11, perfect for low hardware.
  • Custom Repositories — Contains all packages that Void does not have.
